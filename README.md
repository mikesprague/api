# api

Using GH Action Workflows and GH Pages to generate some endpoints with Node.js

## Endpoints

- [![National Day](https://github.com/mikesprague/api/actions/workflows/national-day.yml/badge.svg)](https://github.com/mikesprague/api/actions/workflows/national-day.yml)
  - endpoint: <https://mikesprague.github.io/api/national-day.json>
  - uses data from: <https://nationaldaycalendar.com>
- [![GitHub Trending Repos](https://github.com/mikesprague/api/actions/workflows/github-trending-repos.yml/badge.svg)](https://github.com/mikesprague/api/actions/workflows/github-trending-repos.yml)
  - endpoint: <https://mikesprague.github.io/api/github-trending-repos.json>
  - uses data from: <https://github.com/trending?spoken_language_code=en>
